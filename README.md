# Bhargav Nallan Chakravartula - Portfolio

A responsive personal portfolio focused on semiconductor tapeout, foundry automation, VLSI design, and computational device modeling.

## Files to publish

Upload the complete contents of this folder to the root of the GitHub repository:

```text
index.html
styles.css
script.js
README.md
assets/
  favicon.svg
  Bhargav_Nallan_Resume.pdf
```

The `assets` folder and its contents must remain in that exact relative location because the website links to them.

## Preview locally

Open `index.html` in a browser, or run:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Publish with GitHub Pages

1. Create a public repository named `ra5enagn.github.io` for a root personal site, or use another repository name for a project site.
2. Upload all files and the `assets` folder to the repository root.
3. Open **Settings > Pages**.
4. Select **Deploy from a branch**, branch `main`, folder `/ (root)`.
5. Save the settings.

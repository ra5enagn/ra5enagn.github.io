# Bhargav Nallan Chakravartula — Portfolio

A compact, responsive portfolio for GitHub Pages built entirely with HTML and CSS.

## Files

- `index.html` — content and CSS-only tab controls
- `styles.css` — layout, visuals, responsive design, tabs, and theme toggle
- `assets/` — résumé, favicon, and semiconductor illustration

No JavaScript file is required.

## Publishing

Upload the contents of this folder to the root of the `ra5enagn.github.io` repository and enable GitHub Pages from the `main` branch and `/ (root)` folder.

## Notes

The tab navigation uses hidden radio buttons and CSS sibling selectors. The light/dark toggle is also CSS-only, so the selected theme resets when the page is refreshed.
